# ⚡ EXECO AI Agent

EXECO is an AI-powered Web3 assistant that lets users interact with blockchain using natural language commands.
EXECO is a modern AI-powered Web3 assistant that allows users to interact with blockchain actions using natural language.

Users can send, swap, and interact with smart contracts simply by typing commands like:

> "Send 10 USDC to 0x..."  
> "Swap ETH to USDT"  
> "Switch To Base" 
---

## 🚀 Features

- AI chat-based Web3 interaction  
- Send tokens using simple commands  
- Swap tokens easily  
- Wallet integration (MetaMask / WalletConnect)  
- Multi-chain support (Sepolia default + custom EVM chains)  
- Direct smart contract interaction
